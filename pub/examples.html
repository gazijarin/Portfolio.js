portfolio.js
Portfolio.js is a small JS library that helps with clear and succinct data presentation that is easy to navigate. Built by Gazi Jarin.
examples api get started
examples
about
The About display showcases general information about the user, organization or product. It has a simple title, description, icon and an optional stack that can outline various skillsets.
JS

            const portfolio = new Portfolio();

            const about = portfolio.createAbout(".ex-about-display");
            about.addAboutContent(
              "Hi, I'm Gazi!",
              ["JAVASCRIPT ES6+", "PYTHON", "JAVA", "C#", "HTML/CSS"],
              "I am currently a fourth year Computer Science major at University of Toronto. After graduation, I will be joining Amazon as a Software Development Engineer while undertaking a part-time Master's of Science in Software Engineering at University of Oxford. \nBelow are some languages I am working with currently:",
              "./example-assets/icon2.png"
            );
item display

The ItemDisplay component showcases key information about an item. In this case, we utilized the component to showcase a few side projects.

It has a title, description, image and an optional stack for links. The ItemDisplay component can be customized as an ItemSlider or as an ItemGrid.

experience display

The ExperienceDisplay component showcases key information about the progression of one's experiences.

In this case, we utilized the component to showcase industry experience. It has a title, description, image and an optional stack for skillsets learned.

The ExperienceDisplay component can be customized as an ExpStepper which shows the experience in a timeline fashion or as an ExpTable which shows the experience in tab format. The user has liberty to add as many steps (or tabs) as visually viable.

api
portfolio
METHOD(S) DESCRIPTION
createAbout(location) Creates an About display at location.
createItemDisplay(location) Creates an ItemDisplay at location.
createExpDisplay(location) Creates an ExpDisplay at location.
about display
METHOD(S) DESCRIPTION
getAbout() Returns this About instance.
addAboutContent(title, stack, description, iconUrl) Populates the About instance with title, stack, description and iconUrl.
item display
METHOD(S) DESCRIPTION
getItemDisplay() Returns this ItemDisplay instance.
createItemGrid() Creates an ItemGrid instance.
createItemSlider() Creates an ItemSlider instance.
item grid
METHOD(S) DESCRIPTION
getItemGrid() Returns this ItemGrid instance.
addItem(title, description, iconUrl, stack) Populates the ItemGrid instance with a new Item with title, stack, description and iconUrl.
item slider
METHOD(S) DESCRIPTION
getItemSlider() Returns this ItemSlider instance.
addItem(title, description, iconUrl, stack) Populates the ItemSlider instance with a new Item with title, stack, description and iconUrl.
experience display
METHOD(S) DESCRIPTION
getExpDisplay() Returns this ExpDisplay instance.
createExpStepper() Creates an ExpStepper instance.
createExpTable() Creates an ExpTable instance.
exp stepper
METHOD(S) DESCRIPTION
getExpStepper() Returns this ExpStepper instance.
addExpItem(stepText, stepTitle, stepDesc, stack) Populates the ExpStepper instance with a new ExpItem with stepText, stepTitle, stepDesc and stack.
exp table
METHOD(S) DESCRIPTION
getExpTable() Returns this ExpTable instance.
addExpItem(stepText, stepTitle, stepDesc, stack) Populates the ExpTable instance with a new ExpItem with stepText, stepTitle, stepDesc and stack.
